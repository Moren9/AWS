Tecnologie Utilizzate

  • AWS (Amazon Web Services)
  • Docker
  • HTML5 e CSS3
  • HTTPS con Certificato Self-Signed
  • Form di Login Sicuro

Inizio

Per la realizzazione del compito il primo passo necessario è stato quello di creare una istanza AWS "EC2"; procedendo poi con la configurazione di quest'ultima, inserendo nome, VM e coppia di chiavi utilizzata.

Server

Per la configurazione del server sarà necessario possedere il software PuTTy, in alternativa si può avviare il terminale direttamente dalla scheda di AWS; si procede installando docker in modo tale da consentire la comunicazione tra i diversi container.

Progetto

Nelle, potremmo definirle, fasi finali del lavoro sarà necessario creare le varie cartelle contenute nel progetto, inserire i file contenenti il codice per ogni parte del lavoro di cui avremo bisogno e, per testare, infine, il tutto non basterà altro che inserire l'indirizzo IP pubblico della nostra interfaccia su un qualsiasi browser web.
